# Быстрый старт с AI улучшениями

## Настройка KIE.ai

1. **Добавьте API ключ в `.env`:**
   ```bash
   KIE_API_KEY=ваш_ключ_здесь
   KIE_API_URL=https://api.kie.ai/v1
   ```

2. **Проверьте настройки refinement:**
   ```bash
   REFINEMENT_ENABLED=true
   ```

## Что улучшает AI

### Автоматически улучшается:

1. **Сегментация** (удаление фона)
   - Чистые края
   - Удаление артефактов
   - Плавные переходы

2. **Композитинг** (финальная сборка)
   - Смешивание элементов
   - Подбор освещения
   - Естественные тени
   - Цветовая гармония

### Опционально:

3. **Размещение** (если включено в config)
   - Оптимальная позиция
   - Правильный масштаб

## Использование

### Базовый запуск (с AI):

```bash
python main.py --input dog.jpg --template dog_cosmonaut_v1 --output result.png --debug
```

AI автоматически улучшит результат на этапах:
- После сегментации (если `improve_segmentation: true`)
- После композитинга (всегда, если ключ настроен)

### Настройка в template_config.json:

```json
{
  "refinement": {
    "enabled": true,
    "improve_segmentation": true,  // Улучшить удаление фона
    "improve_placement": false,     // Улучшить размещение (опционально)
    "second_pass": true             // Второй проход для тонкой настройки
  }
}
```

## Проверка работы

1. **Проверьте логи** - должны быть сообщения:
   ```
   INFO - Image refined using KIE.ai via /v1/images/edit
   ```

2. **Проверьте debug файлы:**
   - `03_subject_rgba_ai_improved.png` - улучшенная сегментация
   - `05_refined_pass1.png` - первый проход
   - `05_refined_pass2.png` - второй проход (если включен)

3. **Сравните результаты:**
   - `04_composited.png` - до AI
   - `05_refined.png` - после AI

## Если AI не работает

1. Проверьте ключ: `echo $KIE_API_KEY`
2. Проверьте URL в логах
3. Проверьте формат ответа API (может потребоваться адаптация под ваш API)

## Кастомизация промптов

Можно задать свой промпт в `template_config.json`:

```json
{
  "refinement": {
    "prompt_template": "Ваш детальный промпт для улучшения изображения"
  }
}
```

Или система сгенерирует промпт автоматически через LLM.
