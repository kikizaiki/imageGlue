# Шаблон dog_cosmonaut_v1

## Требуемые файлы

Для работы шаблона необходимо подготовить следующие файлы:

1. **base_clean.png** (1024x1536)
   - Постер без исходной собаки
   - Фон и все элементы, кроме собаки

2. **occlusion_mask.png** (1024x1536, RGBA)
   - Элементы, которые должны перекрывать вставленную собаку
   - Например: части шлема, руки космонавта и т.д.
   - Белый/непрозрачный = перекрывает, черный/прозрачный = не перекрывает

3. **glass_fx.png** (1024x1536, RGBA, опционально)
   - Эффекты: блики на стекле, засветы, шум
   - Может быть пустым (полностью прозрачным)

## Подготовка шаблона

1. Откройте исходный постер в графическом редакторе
2. Удалите/закрасьте исходную собаку → сохраните как `base_clean.png`
3. Выделите элементы перекрытия → сохраните как `occlusion_mask.png`
4. При необходимости создайте эффекты → сохраните как `glass_fx.png`
5. Настройте параметры в `template_config.json`:
   - `insert_zone` - зона вставки собаки
   - `crop_expansion` - расширение кропа
   - `placement` - параметры размещения
   - `postprocess` - параметры постобработки

## Проверка

После подготовки файлов проверьте:

```bash
python main.py --input test_dog.jpg --template dog_cosmonaut_v1 --debug
```

Все промежуточные этапы будут сохранены в `runs/debug/<job_id>/`
